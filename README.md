## General info
 A commercial Android Application to allow multiple on-campus dining options to students.
 Served more than 600 students and and 1700+ orders.
	
## Technologies
Project is created with:
* Firebase Database
* MySQL
* PHP
* SQLite Database
* JAVA
* HTTP RequestManager
